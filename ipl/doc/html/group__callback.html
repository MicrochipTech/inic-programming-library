Detailed Description

The following callback functions need to be implemented in an application and are used by the IPL to interface with the INIC hardware.

uint8_t Ipl_GetIntPin ( void  )

#include <C:/Roland/AppsTeam/Misc/InicProgLib/trunk/ipl/inc/ipl_pb.h>

Hardware abstraction. Callback function to read INIC's INT_ pin status.

Optional. Only required if INT_ pin is connected and should be used. Enabled by IPL_USE_INTPIN.

Returns
Possible result values:
Value Description
0 INT_ pin is LOW
1 INT_ pin is HIGH
2...255 Error occured when INT_ pin was read
