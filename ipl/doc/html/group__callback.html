Detailed Description

The following callback functions need to be implemented in an application and are used by the IPL to interface with the INIC hardware.

Function Documentation

uint8_t Ipl_SetResetPin ( uint8_t  lowHigh)

Hardware abstraction. Callback function to set INIC's RESET_ pin.

Parameters
lowHighPin setting of RESET_
lowHigh Description
0 Pull the RESET_ pin to GND
1 Release the RESET_ pin
Returns
Possible result values:
Value Description
0 No error occured
1...255 Error occured when pin was set

void Ipl_Progress ( uint8_t  percent)

Callback function to deliver progress indication for long running jobs.

Optional. Enabled by IPL_PROGRESS_INDICATOR.

Parameters
percentPercentage of job fullfillment (0-100)
percent Description
0 Long running job has just started
1..99 Long running job is still running
100 Long running job has finished
