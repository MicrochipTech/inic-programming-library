Macros

#define IPL_INICDRIVER_OPENCLOSE
 

Detailed Description

If your application has functions for setting up the INIC (I2C) driver, you can let them call by the IPL via the callback functions Ipl_InicDriverOpen() and Ipl_InicDriverClose().
